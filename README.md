# Lakshyadeep's Portfolio 🚀

A high-performance, visually stunning personal portfolio built with modern web technologies. This portfolio is a demonstration of clean code architecture, engaging aesthetics, and smooth user experience.

## 🛠️ Tech Stack
- **Framework:** React.js (Vite)
- **3D & Canvas:** React Three Fiber (@react-three/fiber), Drei (@react-three/drei)
- **Animations:** GSAP (GreenSock Animation Platform)
- **Styling:** Tailwind CSS

## 🌟 Key Features
- **Interactive 3D Elements:** Features a custom 3D Deadpool mask that perfectly aligns and reacts to cursor movements using React Three Fiber.
- **Extreme Performance:** Optimizations implemented for fast loading times and excellent Lighthouse performance scores.
- **Advanced Animations:** Smooth transitions and scroll-triggered animations powered by GSAP.
- **Dynamic Projects Showcase:** A "Works" section highlighting 5 practical, AI-powered web applications built for real businesses.
- **Responsive Design:** Fluid layout adapting perfectly from mobile screens to desktop monitors.

## 💻 Featured Projects
1. **Code Udaan** - A full production website for a hands-on developer program.
2. **Quizzeria (QuizX Crown)** - A fully interactive quiz platform for competitive events.
3. **Rapid Fire Quiz** - A fast-paced buzzer quiz web app for live college events.
4. **Techera 2K26** - Official registration platform for a college tech fest.
5. **Xenon Arena** - A live esports tournament platform for BGMI and Free Fire.

## 🚀 Getting Started

Clone the repository:
```bash
git clone https://github.com/Lakshyadeep405/Portfolio.git
```

Install dependencies:
```bash
npm install
```

Run the development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```
